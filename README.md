# My AP Computer Science Portfolio

This is my professional portfolio showcasing my programming skills, projects, and growth through AP Computer Science.

## Portfolio Sections

- **Home**: Introduction and welcome
- **About Me**: Interests, hobbies, and technology interests
- **Projects**: Programming projects with descriptions, challenges, and lessons learned
- **Resume**: Education, activities, leadership, and technical skills
- **Reflection**: 300+ word reflection on growth, challenges, and learning

## Technologies Used

- HTML5
- CSS3
- Responsive Design
- GitHub Pages

## Getting Started

1. Clone this repository
2. Edit the HTML files with your personal information
3. Add your projects to the Projects page
4. Push changes to GitHub
5. Visit `https://jhurley2028.github.io` to view your portfolio

## Customization

You can customize the portfolio by:

- Editing HTML files with your information
- Modifying colors in `css/style.css` (look for `:root` variables)
- Adding more projects to the Projects page
- Updating links to your GitHub projects

## Tips for Success

1. **Personalize Everything**: Replace all placeholder text with your actual information
2. **Add Projects**: Include at least 4 of your programming projects with descriptions
3. **Be Specific**: In reflections and project descriptions, be detailed about what you learned
4. **Keep it Professional**: Use formal language and avoid slang
5. **Add Links**: Link to your GitHub repositories for each project
6. **Proofread**: Check for spelling and grammar errors
7. **Test Responsiveness**: Make sure your portfolio looks good on mobile devices

## Grading Criteria

This portfolio is evaluated on:

- **Organization & Design (20 pts)**: Clean, professional layout
- **Completeness (20 pts)**: All required sections included
- **Project Explanations (20 pts)**: Clear descriptions of your work
- **Technical Quality (20 pts)**: Functional, bug-free code
- **Reflection & Professionalism (10 pts)**: Thoughtful reflection, formal tone
- **Creativity & Effort (10 pts)**: Unique touches and polish
